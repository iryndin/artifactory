This is the Artifactory Maven2 Repository Update Manager
 and
Artifactory Command Line Admin Tool.

For more information about Artifactory, please visit:

   http://www.jfrog.org/sites/artifactory/latest/

RUNNING ARTIFACTORY UPDATE MANAGER
==================================
From the extracted zip, you can run the updater with:
Unix:
   ./bin/artdump
Windows:
   bin\artdump.bat

RUNNING ARTIFACTORY COMMAND LINE ADMIN
======================================
From the extracted zip, you can run the artadmin with:
Unix:
   ./bin/artadmin
Windows:
   bin\artadmin.bat
