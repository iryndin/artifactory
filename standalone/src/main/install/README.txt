This is the Artifactory Maven2 Repository Proxy and Manager.

For more information about Artifactory, please visit:

   http://www.jfrog.org/sites/artifactory/latest/

The server comes with an embedded version of Jetty.
Information about running and configuring Jetty can be found on the Jetty wiki:

   http://docs.codehaus.org/display/JETTY/


RUNNING ARTIFACTORY
===================
From the release, you can run the server with:

   ./bin/artifactory.sh
   or
   bin\artifactory.bat

and then point your browser at

   http://localhost:8081/artifactory/

The default administrator user created on the first run is:
username: admin
password: password
