

This is the Artifactory Maven2 Repository Proxy and Manager.

For more information about Artifactory, please see the jforg site:

   http://www.jfrog.org/sites/artifactory/latest/

The server comes with jetty embedded, and running, configuring,
and dependencies of jetty can be found on jetty wiki:

   http://docs.codehaus.org/display/JETTY/


RUNNING ARTIFACTORY
===================
From the release, you can run the server with:

   ./bin/artifactory.sh
   or
   bin\artifactory.bat

and then point your browser at

   http://localhost:8081/artifactory/

the default user created is username=admin password=password
